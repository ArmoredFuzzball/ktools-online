Online ktools for Don't Starve Together

Version 1.1.0

This project can convert compiled images and animations from Don't Starve Together
and convert them back into pictures and Spriter projects for use in modding.

Original ktools by: nsimplex
This project uses a ktools fork by: victorpopkov (commit 186)
And this project is developed by: ArmoredFuzzball
Project Github

TEX to PNG (~10MB max)


Will download a png when finished.

Anim zip to Spriter project (~10MB max)

Please ensure there is a build.bin and anim.bin (and tex atlases, if there are any) in your zip!

Will download a zip when finished.
